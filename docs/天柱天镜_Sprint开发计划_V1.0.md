## 完成标准（全局 Definition of Done）

以下条件适用于所有 Story，除非 Story 中明确豁免：

1. **代码**：功能代码已合并到 `develop` 分支，PR 经 ≥1 人 Code Review 通过
2. **测试**：单元测试覆盖率 ≥ 80%（Service 层），集成测试通过
3. **文档**：接口变更已同步 API 规范；数据库变更已同步数据库设计文档
4. **安全**：无 SQL 注入风险、日志无敏感信息输出、接口有权限注解
5. **日志**：关键业务操作有含 `trace_id` 的 INFO 日志
6. **验收**：Story 验收标准中所有验收项 ✅ 全部通过
