- **影响范围**：所有页面联调（级联阻断）
- **现状**：`AuthController` 只有 `/login`、`/refresh`、`/logout`，无 `/me` 端点

### ✅ BLK-02：离线仿真模块路径冲突 + 上传方式不符【已修复 2026-04-01】

- **影响范围**：离线仿真模块全部接口（4 个）
- **路径冲突**：
  - 后端：`@RequestMapping("/api/v1/simulation")`（单数）
  - 前端：调用 `/simulations`（复数，符合规范）

- **上传方式不符**：

  - 后端：`GET /simulation/upload-url`（预签名 URL 方式）
  - 结果：文件上传逻辑完全不通
- **文件**：
  - `services/history-replay-service/src/main/java/com/tianzhu/tianjing/history/controller/SimulationController.java`
