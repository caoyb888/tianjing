## 算法类型说明

| 类型 | 含义 |
|---|---|
| DETECTION | 目标检测，输出边界框与类别 |
| SEGMENTATION | 语义/实例分割，输出像素级掩码 |
| CLASSIFICATION | 图像分类，输出类别标签 |
| MEASUREMENT | 几何测量，输出尺寸数值 |
| ENHANCEMENT | 图像增强/预处理，作为前置算子使用 |
