## 1. 钢材表面缺陷数据集

### 1.1 NEU Surface Defect Database

| 项目 | 内容 |
|---|---|
| **来源机构** | 东北大学（NEU）|
| **下载地址** | http://faculty.neu.edu.cn/yunhyan/NEU_surface_defect_database.html |
| **规模** | 1,800 张，每类 300 张，分辨率 200×200 |
| **标注格式** | 图像分类（含目标检测版本） |
| **许可证** | 学术研究免费使用 |

**类别（6 类）：**

| 英文名 | 中文名 | 典型特征 |
|---|---|---|
| Rolled-in Scale | 轧入氧化皮 | 不规则黑色斑点 |
| Patches | 斑块 | 大面积色差区域 |
| Crazing | 开裂 | 网状细裂纹 |
| Pitted Surface | 点蚀表面 | 密集小坑洞 |
| Inclusion | 夹杂物 | 点状或线状异物 |
| Scratches | 划痕 | 线性划伤痕迹 |

## 4. 使用建议

### 4.1 推荐训练策略（两阶段微调）

```
阶段一：领域预训练
  基础权重：yolov8n.pt（COCO 预训练）
        ↓
  用 GC10-DET / NEU / Severstal 继续训练
  目标：让模型"看懂"钢铁表面，而非 COCO 通用场景
        ↓
  产出：steel_pretrained.pt（钢铁领域权重）

阶段二：场景微调
  基础权重：steel_pretrained.pt（阶段一产出）
        ↓
  用平台标注的自有数据集（仿真任务→生成数据集）
  目标：识别具体场景的异常类型
        ↓
  产出：scene_specific.onnx（上线推理模型）
```

### 4.2 数据量参考

| 数据量 | 预期效果 |
|---|---|
| < 50 张/类 | 过拟合严重，仿真可能有效但生产场景失效 |
| 100–300 张/类 | 基本可用，建议结合数据增强 |
| 300–1000 张/类 | 工业场景可接受精度（mAP50 > 0.80） |
| > 1000 张/类 | 生产可靠（mAP50 > 0.90） |

### 4.3 训练超参建议（修改平台 TRAIN_CONFIG_JSON）

```json
{
  "epochs": 100,
  "img_size": 640,
  "batch_size": 16,
  "augment": true
}
```

> 当前默认 epochs=10 对工业场景严重不足，建议至少 50 轮。

## 5. 格式转换说明

### 5.1 平台要求的数据集格式

上传到 `tianjing-datasets` bucket 的 `dataset.zip` 需满足：

```
dataset.zip
├── images/
│   ├── 00001.jpg
│   ├── 00002.jpg
│   └── ...
└── annotations.json   # COCO 格式
```

### 5.2 各数据集转换方法

**GC10-DET（YOLO → COCO）**：

```bash
# 使用 ultralytics 自带工具转换
pip install ultralytics
python3 -c "
from ultralytics.data.converter import convert_coco
# 先将 YOLO 标签转为 COCO JSON
"
# 或使用开源工具：https://github.com/ultralytics/JSON2YOLO（双向转换）
```

**Severstal（RLE掩码 → BBOX）**：

```python
import pandas as pd
import numpy as np

def rle_to_bbox(rle_string, height, width):
    """将 RLE 掩码转换为 BBOX (x_min, y_min, x_max, y_max)"""
    if pd.isna(rle_string):
        return None
    nums = list(map(int, rle_string.split()))
    pixels = []
    for i in range(0, len(nums), 2):
        start = nums[i] - 1
        length = nums[i + 1]
        pixels.extend(range(start, start + length))
    pixels = np.array(pixels)
    rows = pixels % height
    cols = pixels // height
    return int(cols.min()), int(rows.min()), int(cols.max()), int(rows.max())
```

**VOC XML → COCO JSON**：

```bash
# 使用 roboflow 或 labelImg 自带转换功能
# 或使用：https://github.com/Tony607/voc2coco
python3 voc2coco.py --ann_dir ./Annotations --output result.json
```

### 5.3 上传到平台

```bash
# 打包并上传数据集到 MinIO（需在服务器上执行）
cd /path/to/dataset
zip -r dataset.zip images/ annotations.json

uv run --with minio python3 - <<EOF
from minio import Minio
client = Minio("localhost:9000",
               access_key="minioadmin",
               secret_key="minioadmin123",
               secure=False)
client.fput_object(
    "tianjing-datasets",
    "DV-GC10-STEEL-V1/dataset.zip",   # 修改为你的数据集版本 ID
    "dataset.zip",
    content_type="application/zip"
)
print("上传完成")
EOF
```
