在《天柱·天镜》V2.0 方案中，为了避免“烟囱式”的独立开发，平台规划了**“原子化算法仓”**架构，通过共享特征主干网（Backbone）并替换任务头（Task Head）来实现算法复用。

针对梳理出的 16 个业务场景，我为你筛选并映射了相应的底层算法模型，重点平衡了工业现场的极端环境抗干扰能力与边缘/服务端的推理实时性：

### 工业视觉场景与算法选型矩阵

|**序号**|**厂部及工序**|**业务场景**|**推荐算法模型**|**算法简要描述**|**适合的业务维度**|
|---|---|---|---|---|---|
|**1**|球团 - 链篦机|侧板跑偏检测|**RT-DETR + 亚像素边缘提取**|RT-DETR 提供端到端的高精度目标检测（无需 NMS），结合 Canny 或亚像素级边缘检测算法计算偏移量，满足 10mm 精度要求。|设备状态监测（保停机风险）|
|**2**|球团 - 分料器|运行状态检测|**EfficientNetV2 / ResNet50**|轻量级图像分类网络，用于判别分料器的开/关/半开状态，推理极快，适合部署在边缘端。|设备状态监测|
|**3**|球团 - 成一皮带|料面检测|**YOLOv9-Seg / Mask2Former**|实例分割/语义分割模型，用于精准提取皮带上物料的轮廓面积，结合相机标定估算料面高度与堵料趋势。|工艺参数监控|
|**4**|烧结 - 看火系统|点火料面（点火强度）|**Retinex 增强 + 分割模型**|先通过 Retinex 算法抑制强光/光晕，再利用分割模型提取有效燃烧面积，量化点火强度。|工艺参数监控|
|**5**|烧结 - 烧结机|壁条脱落/堵塞/翘起|**YOLOv9 / YOLOv10**|擅长多尺度目标检测，结合平台的连续帧确认逻辑，能够有效抵抗粉尘干扰，快速检出细小目标（脱落或翘起的壁条）。|设备状态监测|
|**6**|烧结 - 二配室|白灰/燃料仓粒度|**SAM (Segment Anything) 微调**|零样本/少样本泛化能力极强的分割大模型，微调后可对堆叠的白灰和燃料进行高精度的单体分割，用于计算粒径分布。|工艺参数监控（P4 攻关）|
|**7**|烧结 - 台车|台车篦条检测|**YOLOv9**|运动目标检测，适用于 135 部台车连续运动中的缺损快速抓拍。|设备状态监测|
|**8**|炼钢 - 转炉车间|溅渣渣粒密度识别|**密度估计网络 (CSRNet) / 分割**|专门处理密集目标的特征提取网络，将渣粒画面映射为密度图（Density Map），量化渣粒浓度以指导溅渣枪位。|工艺参数监控（P4 攻关）|
|**9**|炼钢 - 吹氩站|底吹处理|**DCP (暗通道先验) + 分割模型**|必须先利用 DCP 或深度学习去雾算法去除强烟气遮挡，再使用分割网络识别钢水裸露面积以调整氩气流量。|工艺参数监控（P4 攻关）|
|**10**|炼钢 - 连铸|铸坯表面缺陷识别|**YOLOv9 / RT-DETR**|核心缺陷检测，针对裂纹、夹渣、气泡等多种形态不一的表面缺陷，具备良好的鲁棒性。|质量检测|
|**11**|型钢 - 加热炉|铸坯质量|**YOLOv9 (多尺度特征融合)**|与场景 10 共享主干网络，通过增强 FPN 结构抵抗氧化铁皮带来的纹理干扰。|质量检测|
|**12**|型钢 - 冷床入口|钢材质量|**YOLOv9 (多任务头)**|多角度相机协同检测，针对翼缘弯曲、表面裂纹等缺陷进行分类与定位。|质量检测|
|**13**|型钢 - 冷床入口|钢材尺寸|**双目/线结构光 3D 算法 + 边缘提取**|纯 2D 视觉难以满足高精度钢材尺寸测量，建议采用线结构光结合 3D 点云提取算法，或亚像素 2D 边缘拟合。|质量检测|
|**14**|型钢 - 检查台架|钢材质量|**YOLOv9**|成品红检复核，与场景 12 共享底层算法网络，重点降低极低容忍度下的误报率。|质量检测|
|**15**|带钢 - 飞剪|飞剪优化剪切|**Keypoint Detection (关键点检测)**|极速推理场景（<10ms闭环），利用轻量级姿态或关键点检测网络，精准定位带钢头部形状与坐标，配合热检计算切头长度。|工艺参数监控|
|**16**|带钢 - 卷取|端面质检|**ResNet50 / 异常检测 (Padim)**|对于端面折叠、塔形等全局性特征，图像分类器或无监督异常检测算法能提供比目标检测更稳定的判别。|质量检测|
